# 📈 Zerodha Clone

## 📖 Project Overview

Zerodha Clone is a stock trading dashboard inspired by Zerodha. It displays holdings, positions, orders, and portfolio information.

---

## 🚀 Features

- Dashboard
- Holdings
- Orders
- Positions
- Portfolio Summary
- Responsive UI

---

## 🛠 Tech Stack

- React
- JavaScript
- HTML
- CSS
